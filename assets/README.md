# Assets

跨应用共享素材区。

| 路径 | 内容 |
|---|---|
| `icons/` | Portal、Command Center、文档页共用图标 |
| `creative/cover-backgrounds/` | 封面背景图 |
| `creative/reference-images/` | 角色、UI、视觉参考图 |
| `creative/photoshop-templates/` | PSD 比例模板 |

## 维护规则

- 素材文件可以保留易识别的原始名称。
- 代码和目录名优先使用小写 kebab-case。
- 大素材只放真正会复用的版本；临时截图、导出草稿不要提交到仓库。
