# Source Log

> 本文件记录剧情资料库 v1 参考的信息来源类型。后续维护时，建议把关键来源补到对应版本文件末尾。

## 官方来源优先级

1. 游戏内主线剧情、角色任务、活动剧情
2. 官方版本更新公告
3. 官方 PV / 前瞻直播 / 角色展示
4. 官方网站世界观与角色介绍
5. 官方社媒补充设定

## 社区来源使用原则

社区 wiki、剧情录屏、考据视频可以用于查漏补缺，但不要直接当作唯一事实来源。  
涉及“推测”“内鬼”“测试服变化”的内容必须标注来源性质。
