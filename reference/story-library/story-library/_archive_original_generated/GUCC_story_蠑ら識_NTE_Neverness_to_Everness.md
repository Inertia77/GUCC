## 0. 最短版先记住

《异环》是 Hotta Studio 开发的超自然都市开放世界 RPG。故事从**海特洛市**开始。玩家作为首位“无证上岗”的**异象猎人**，加入接取民间异象委托维持周转的古董店**伊波恩**，与伙伴们调查城市里的异常事件。

这个世界的特点不是“怪物突然入侵人类城市”，而是：

```text
人类
  +
异能者
  +
歧骸等超自然存在
  +
异象管理局等官方机构
  +
民间猎人组织与古董店伊波恩
  +
誓约、环、预言等深层规则
```

它的叙事味道更像：

> **都市日常 + 怪谈委托 + 超自然政治 + 轻喜剧外壳下的大谜团。**

---

## 1. 世界观底层：海特洛市、异象、异能者、歧骸

### 1.1 海特洛市

海特洛市是玩家进入故事的第一站。  
它不是传统意义上的安全城市，而是一个“异常已经融入日常”的都市：

- 异象管理局大厅里有排队申领居民许可证的歧骸；
- 异能者像普通上班族一样赶电车；
- 街头可能在抓捕团三郎；
- 管家、维修师、古董店、照相馆、神秘当铺都可能和异常有关；
- 城市生活的搞笑日常背后有“环”和预言的阴影。

所以海特洛市不是“怪谈发生地”，而是**怪谈已经制度化、生活化的城市**。

### 1.2 异象

异象是这个世界的核心事件类型。  
它既可以是小型都市怪谈，也可能牵出更大规则：

- 雨人活跃期；
- 神秘当铺；
- 异象空间幽灵号；
- 团三郎抓捕；
- 誓约与环相关现象；
- 支离现世中的预言回响。

异象的作用相当于《绝区零》的空洞委托入口，但风格更轻奇幻、更都市怪谈。

### 1.3 异能者与歧骸

公开资料中出现了“异能者”“歧骸”等存在。  
它们说明这个世界不是人类独占城市，而是多种异常存在共同生活。

这带来的社会问题包括：

```text
谁算居民？
谁需要许可证？
异能者是否受普通法律约束？
歧骸能否正常就业、消费、社交？
异象猎人和官方管理局边界在哪里？
```

这也是异环未来剧情很可能展开的地方：**超自然存在如何被城市治理吸收。**

## 2. 主角定位：无证上岗的异象猎人

玩家身份是“首位无证上岗的异象猎人”，加入古董店**伊波恩**。

这个设定很关键，因为它天然带有灰色感：

```text
异象管理局 = 官方治理
伊波恩 = 民间委托 / 古董店 / 半灰色调查据点
主角 = 没有正式执照但能解决问题的人
```

这和《绝区零》的法厄同有一点像：都是在官方体系之外处理异常。  
但区别是：

```text
法厄同：非法绳匠，处理空洞导航，追查旧都真相
异环主角：无证异象猎人，处理都市异常，逐渐接触环与誓约
```

一个是灾害工业城市的地下导航员；  
一个是怪奇都市的民间异常调查员。

---

## 3. 伊波恩古董店

伊波恩是当前公开剧情中最重要的主角据点。  
它接取民间异象委托，靠处理各种异常事件维持周转。

伊波恩的叙事功能：

- 给玩家一个城市生活据点；
- 把小委托串成主线；
- 容纳性格各异的伙伴；
- 连接普通居民、异常存在、官方机构、猎人组织；
- 作为“日常喜剧”和“大阴谋”之间的转换器。

可以这样理解：

```text
伊波恩
  = 主角家
  = 委托入口
  = 队伍关系中心
  = 城市怪谈收集站
  = 通往环与预言的第一层门
```

## 4. 当前公开剧情线索

由于异环公开主线文本仍有限，当前能稳定整理的剧情线索如下。

## 4.1 海特洛都市日常线

这是第一层。  
玩家在城市中处理各种奇怪但带生活味的事件：

- 神秘当铺只在特定时间出现；
- 塔吉多恋情危机；
- 照相馆恢复营业；
- 雨人进入活跃期；
- 街头追捕团三郎；
- 和各种居民、异能者、歧骸建立关系。

这一层的作用是建立城市亲切感。  
异环不是一上来就世界毁灭，而是让玩家相信这个城市“值得逛、值得认识、值得守”。

---

## 4.2 异象委托线

第二层是伊波恩处理民间委托。  
这些委托看似轻松，但会逐渐暴露规则：

```text
普通居民的麻烦
  ↓
牵出异常现象
  ↓
发现官方记录不足或无法插手
  ↓
伊波恩介入
  ↓
主角和伙伴建立信任
```

这套结构很适合单元剧：

- 一集一个都市怪谈；
- 每集解决一个人情问题；
- 长线慢慢推进“环”和“誓约”。

---

## 4.3 誓约线

官方商店介绍明确提到“牢不可破的誓约伟力”。  
誓约很可能是异环世界的重要超自然规则。

目前可以先这样理解：

```text
誓约
  = 某种约束现实/个体/异象的契约型力量
  = 可能和角色能力、事件因果、城市规则有关
  = 后续主线大概率会解释代价与破约后果
```

视频选题上，誓约是异环最值得长期追踪的关键词之一。

---

## 4.4 环与预言线

官方介绍还提到：

- “环”这一庞然大物的身影；
- 支离现世的“预言”回响。

这说明“异环”这个标题不是单纯好听。  
“环”很可能是世界结构、异常源头或终极危机的一部分。

当前稳妥判断：

```text
都市小异象
  ↓
誓约规则
  ↓
预言回响
  ↓
环
  ↓
世界级真相
```

也就是说，异环主线可能会从“今天帮居民解决怪事”逐渐升级成“这个城市为什么会和异常共生”。

```text

  ↓ 人类与异常共存的怪奇都市

  ↓ 无证上岗，解决民间异常事件

异能者
  ↓ 拥有特殊能力的人群，已融入城市日常

歧骸
  ↓ 超自然/非人居民，需要居民许可证等管理

猎人组织
  ↓ 各有特色，可能与伊波恩合作或竞争

环 / 誓约 / 预言
  ↓ 城市深层规则与主线谜团
```

## 7. 与同类都市二游的区别

### 和《绝区零》

```text
绝区零：灾害后城市，空洞是资源与污染，绳匠处理高危导航。
异环：怪奇共生城市，异象融入日常，猎人处理超自然委托。
```

绝区零更“工业灾害 + 都市地下”。  
异环更“都市怪谈 + 生活喜剧 + 超自然共存”。

### 和《鸣潮》

```text
鸣潮：文明被世界级灾害反复撕裂，主线史诗感更重。
异环：城市本身就是舞台，日常事件也能成为怪谈入口。
```

鸣潮是灾后史诗。  
异环是怪奇都市剧。

### 和《崩铁》

```text
崩铁：每站一个文明命题，星神命途宏大叙事。
异环：从一座城市的人情委托开始，逐渐摸到世界规则。
```

崩铁是银河寓言。  
异环是都市单元剧起步。

## 9. 视频选题钩子

- 《异环》真正卖点不是开放世界，而是“怪奇都市日常”
- 为什么“无证上岗”这个设定很聪明：主角天然站在官方与民间之间
- 伊波恩古董店可能是异环的真正叙事发动机
- 歧骸居民许可证说明了什么：异常已经被城市治理吸收
- “环”和“誓约”可能怎么串起异环主线
- 异环和绝区零都是都市二游，但底层气质完全不同
