# 无限大 / ANANTA 剧情总览

## 0. 快速结论

《无限大 / ANANTA》是高自由度都市开放世界 RPG。当前公开资料重点展示 Nova City、不同角色切换、新任队长、特殊行动小队、都市生活与超常事件。剧情公开仍有限，因此本文档以预研和资料框架为主。

## 1. 世界观入口

```text
Nova City：主要城市舞台。
新任队长：玩家视角/核心身份。
特殊行动小队：主角需要重建或带领的团队。
都市自由行动：驾驶、跑酷、战斗、生活活动。
超常/怪异事件：推动小队行动的主要事件类型。
```

## 2. 与异环区别

无限大是 ANANTA，重点是 Nova City、新任队长、特殊小队和高自由都市动作。  
异环是 NTE，重点是海特洛市、异象猎人、伊波恩古董店和超自然委托。
