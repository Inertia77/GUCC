# 明日方舟：终末地 剧情总览

## 0. 快速结论

《明日方舟：终末地》的叙事重点不再是泰拉大陆上的移动城邦与感染者政治，而是转向塔卫二上的开拓、工业建设、侵蚀威胁和组织化生存。它继承《明日方舟》的冷硬世界观，但表达方式更偏“异星开荒 + 工业秩序 + 即时战术”。

## 1. 世界观入口

```text
地点：塔卫二。
组织：终末地工业。
主角：管理员。
核心伙伴：佩丽卡等终末地成员。
核心威胁：侵蚀、锚点、未知生态/文明遗留。
核心行动：探索、建设、消除侵蚀、发展地区。
```

## 2. 与明日方舟本篇的关系

终末地不是简单换皮续作。它把“罗德岛式危机应对”扩展成“工业组织在异星开拓中建立秩序”。如果说本篇强调感染者、人道、移动城邦政治，那么终末地更强调：

- 一个组织如何在未知土地建立生产线。
- 人类如何面对侵蚀与异星环境。
- 管理员作为中心人物为何被唤醒。
- 工业建设是否也会复制旧世界的问题。
