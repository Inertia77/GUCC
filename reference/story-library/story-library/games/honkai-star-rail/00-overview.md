# 崩坏：星穹铁道 剧情总览

## 0. 快速结论

《崩坏：星穹铁道》的主线不是简单的“列车旅行记”，而是一条围绕「开拓」命途展开的银河政治与神话叙事。每一站都在回答同一个问题：当星神、命途、灾害和制度把普通人压进既定轨道时，人还能不能选择自己的方向？

主线可以分成六个大阶段：

```text
黑塔空间站：开拓者诞生，星核进入身体。
雅利洛-VI：存护世界的封闭与贝洛伯格秩序重建。
仙舟罗浮：丰饶与巡猎的千年仇恨，星核猎手介入。
匹诺康尼：梦境乐园、同谐秩序、钟表匠遗产与开拓精神。
翁法罗斯：黄金裔、火种、轮回、权杖和铁墓。
二相乐园：欢愉、幻造种、告死魔、姬子身份与开拓继承。
```

## 1. 主线骨架

星穹列车继承「开拓」意志，航行在不同世界之间。列车组看似是旅客，实际上每到一站都要面对一个世界的“结构性困境”：

- 雅利洛-VI 的困境是存护变成封闭，牺牲真相换取秩序。
- 仙舟罗浮的困境是长生与仇恨，丰饶赐福变成文明病灶。
- 匹诺康尼的困境是梦境里的幸福，是否值得用自由交换秩序。
- 翁法罗斯的困境是英雄史诗被写进毁灭实验，努力本身也可能成为灾难进度条。
- 二相乐园的困境是想象力、愿力、创伤消费与身份继承。

## 2. 星神与命途关系

星铁世界观的关键是：星神不是普通神明，而是某种极端理念的宇宙化。命途不是职业，而是宇宙中能被践行的道路。

```text
开拓：走向未知，连接世界。
存护：保护、筑墙、延续。
巡猎：复仇、追击、清算丰饶孽物。
丰饶：生命、不死、赐福，也可能失控成灾。
同谐：秩序化的共鸣，也可能压扁个体差异。
欢愉：游戏、荒诞、打破严肃性。
毁灭：终结、崩坏、把世界推向归零。
智识：计算、理解、分类、推演宇宙。
记忆：保存经历，让存在免于被彻底抹去。
```

## 3. 叙事主题

星铁最强的地方不是设定多，而是每个篇章都把“命途”落到人物选择上。

- 布洛妮娅不是因为知道真相就能救国，而是必须承担谎言之后的秩序修补。
- 丹恒不是换个形态就能摆脱前世，而是要重新决定自己如何面对旧债。
- 星期日不是单纯反派，他是在问“如果自由只会带来痛苦，那被温柔安排是否也是幸福？”
- 白厄不是胜利者，他是用无数次失败拖住毁灭的人。
- 姬子线最狠的问题是：如果现在的姬子继承了另一个姬子的愿望，她是否仍然是真正的姬子？

## 4. 管理建议

崩铁文档需要分两层维护：

1. **星球/大篇章层**：雅利洛、仙舟、匹诺康尼、翁法罗斯、二相乐园。
2. **命途/人物主题层**：开拓、存护、丰饶、同谐、毁灭、记忆、欢愉。

做视频时不要只按版本讲，最好按主题切：例如“丰饶为何总是变成灾难”“开拓者为什么是变量”“姬子真假问题其实在讲继承”。
