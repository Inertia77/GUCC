# 崩坏：星穹铁道 人物关系总览

## 1. 列车组

```text
开拓者：星核载体，开拓命途的新变量。
三月七：记忆缺失者，后续与记忆/时间线深度相关。
丹恒：持明旧债与自我重塑的核心人物。
姬子：列车领航员，二相乐园篇后身份问题被重新打开。
瓦尔特：前文明经验者，负责把崩坏式灾难经验带进星铁叙事。
```

列车组不是传统勇者队。它更像一个移动的“命途实验室”：每个人都背着过去，但他们选择继续往前。

## 2. 星核猎手

```text
艾利欧：剧本书写者。
卡芙卡：执行剧本的人，也是开拓者诞生的关键推手。
刃：丰饶诅咒与不死痛苦的人格化。
银狼：现实系统的黑客，常在复杂事件中提供外部突破口。
```

星核猎手不是列车组的正统盟友，但他们推动了很多“必要的相遇”。他们像在未来结果上倒推现在行动的人。

## 3. 翁法罗斯核心人物关系

```text
白厄：负世者，三千多万次轮回中拖住铁墓的人。
昔涟 / 德谬歌 / 迷迷：记忆、爱、第十三泰坦与闭环关键。
来古士：赞达尔思想碎片，权杖实验操盘者。
铁墓：毁灭方程式终点，宇宙级威胁。
阿格莱雅：奥赫玛秩序核心，黄金裔组织者。
刻律德菈：执棋的君主，律法火种，城邦权力与战争布局线。
海瑟音 / 海列屈拉：奏浪的剑骑，海洋火种，与刻律德菈线紧密绑定。
那刻夏：理性怀疑者，撬开神话叙事的工具人也是思想角色。
风堇：天空/医者线，和长夜、治疗、守护相关。
遐蝶：死亡/冥河线，承担死亡规则的人格化表达。
万敌：纷争线，战斗与血统牺牲的代表。
```

翁法罗斯不能只讲白厄和昔涟。刻律德菈、海瑟音、阿格莱雅这一组承载的是“城邦如何把英雄史诗变成制度、战争和秩序”的中层叙事。

## 4. 二相乐园核心人物关系

```text
少女姬子：愿望的源头，可能永远留在二相乐园。
列车姬子：愿望的继承者，可能是幻造种形式的“另一个姬子”。
旧告死魔：十五年前血案源头。
满愿：受害者变成加害者，强制幸福路线代表。
归寂：毁灭旧账与姬子身份攻击的真正危险。
```

二相乐园篇的核心不是“姬子真假”，而是“继承是否能成为真实”。
