# 崩坏：星穹铁道剧情资料库

> 英文名：Honkai: Star Rail  
> 用途：开拓主线、版本剧情、角色关系、阵营与视频选题参考  
> 整理日期：2026-06-27

这个分册用于整理《崩坏：星穹铁道》的剧情脉络。重点放在版本主线、星神与命途、星核事件、仙舟与匹诺康尼等长期线索上，适合在写视频脚本前快速找背景、时间线和可延展选题。

## 快速入口

| 文件 | 内容 |
|---|---|
| `00-overview.md` | 世界观总览、核心概念和阅读顺序 |
| `01-timeline.md` | 主线时间线与版本事件节点 |
| `02-factions.md` | 阵营、势力、组织关系 |
| `03-characters.md` | 角色关系、动机和可用于脚本的观察点 |
| `04-mysteries.md` | 伏笔、悬念、未解问题 |
| `05-video-hooks.md` | 视频标题、切入角度和选题钩子 |

## 使用建议

先读 `00-overview.md` 建立总览，再按 `versions/` 找对应版本。做人物向内容时优先看 `03-characters.md`，做考据或预测向内容时优先看 `04-mysteries.md` 和 `05-video-hooks.md`。

## 维护规则

新增版本资料放进 `versions/`。确定事实、推测和个人判断要分开写；未确认内容请标注为“待验证”或“推测”，避免后续做视频时把临时想法当成定论。
