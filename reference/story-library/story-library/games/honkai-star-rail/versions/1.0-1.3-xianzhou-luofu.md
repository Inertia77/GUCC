# 1.0-1.3 仙舟罗浮：丰饶、药王秘传与星核猎手

## 1. 快速结论

罗浮篇把星铁从单星球政治推向宇宙级命途冲突。丰饶不是简单的治疗，不死也不是祝福；当生命失去死亡边界，文明会被长生反噬。

## 2. 局势

卡芙卡引导列车来到罗浮。星核、药王秘传、建木复苏、幻胧潜入，让罗浮内部危机爆发。丹恒被迫面对饮月旧身和持明旧债。

## 3. 人物关系

```text
景元：用懒散外表遮住长线布局的将军。
丹恒/饮月：前世罪责与今生自我切割。
刃：丰饶不死痛苦的极端样本。
卡芙卡：推动罗浮与列车必要相遇。
幻胧：毁灭令使，利用丰饶与罗浮内部裂缝。
```

## 4. 主题

罗浮篇的主题是“长生的代价”。巡猎与丰饶不是普通善恶，而是一场跨越文明尺度的复仇与清算。
