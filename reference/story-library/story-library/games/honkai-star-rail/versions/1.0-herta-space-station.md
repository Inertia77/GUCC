# 1.0 黑塔空间站：星核载体与开拓起点

## 1. 快速结论

黑塔空间站不是普通新手教程，而是星铁主角存在方式的起点：开拓者不是“出生”，而是被卡芙卡和银狼放入现实舞台；星核不是外挂，而是长期主线炸弹。

## 2. 剧情逻辑

空间站遭遇反物质军团袭击，卡芙卡与银狼行动，开拓者苏醒并成为星核载体。黑塔、艾丝妲、阿兰等人展示智识侧空间站的日常与危机处理。列车组介入后，开拓者被引向星穹列车。

## 3. 人物关系

```text
卡芙卡/银狼 → 制造开拓者入局
黑塔空间站 → 提供智识背景和危机现场
列车组 → 接住开拓者，给予“继续前进”的身份
开拓者 → 从星核容器转变为旅客
```

## 4. 主题

这章最重要的问题是：一个被安排进剧本的人，能不能在旅途中拥有自己的选择？
