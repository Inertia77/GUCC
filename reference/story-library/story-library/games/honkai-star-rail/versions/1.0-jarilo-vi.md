# 1.0 雅利洛-VI：存护、筑城者与星核寒潮

## 1. 快速结论

雅利洛-VI 是星铁第一套完整世界命题：存护原本是保护，但当保护变成封闭、谎言和阶层隔离时，文明会被自己筑的墙困死。

## 2. 局势

贝洛伯格是寒潮中的最后城市。上层区维持秩序，下层区被封锁牺牲。可可利亚受星核影响，试图用极端方式完成“新世界”。

## 3. 主线

列车组进入贝洛伯格，先接触筑城者秩序，再进入下层区理解被牺牲者视角。布洛妮娅在真相、母亲、城市未来之间被迫选择。最终可可利亚失败，布洛妮娅接过大守护者责任。

## 4. 人物关系

```text
可可利亚：被星核蛊惑的旧秩序守护者。
布洛妮娅：新秩序继承者，必须背负真相之后的治理。
希儿：下层区反抗与现实痛苦。
杰帕德/佩拉/希露瓦：贝洛伯格不同层面的守护者。
```

## 5. 主题

雅利洛篇的重点不是“打败坏母亲”，而是“一个城市如何在真相会撕裂秩序时继续活下去”。
