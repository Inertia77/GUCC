# 2.0-2.3 匹诺康尼：梦境、家族、钟表匠与开拓遗产

## 1. 快速结论

匹诺康尼是星铁最完整的“幸福牢笼”篇。它问的是：如果现实痛苦，梦境温柔，那么人是否应该被允许永远睡下去？

## 2. 局势

匹诺康尼表面是盛会之星，游客进入梦境享乐。背后有家族、钟表匠遗产、同谐秩序、公司旧账和各方势力暗战。

## 3. 主要人物

```text
星期日：想用秩序和梦境保护所有人，但代价是自由。
知更鸟：同谐理想中更温柔也更尊重个体的一面。
砂金：公司赌徒，表面轻佻，内核是被命运逼到赌桌上的人。
黄泉：虚无与自我道路的关键人物。
流萤：生命短暂者对现实的执着。
米沙/钟表匠：开拓遗产和梦境真相的入口。
```

## 4. 主题

匹诺康尼不是反快乐，而是反“被安排的快乐”。真正的开拓不是逃进无痛梦境，而是醒来后仍选择向前。
