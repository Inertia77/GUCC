# 3.0-3.7 翁法罗斯：黄金裔、火种、轮回与铁墓

## 1. 快速结论

翁法罗斯表面是希腊神话风的逐火救世史诗，底层是权杖 δ-me13 演算出的记忆/数据世界。黄金裔以为自己在夺火救世，但再创世本身可能推动绝灭大君「铁墓」完成。白厄和昔涟反复轮回，是为了把毁灭拖在失败里。

## 2. 主线结构

```text
第一层：黑潮污染，泰坦失常，黄金裔夺取火种。
第二层：火种试炼让黄金裔成为半神，但也推进实验。
第三层：来古士操控权杖，试图培育铁墓。
第四层：白厄与昔涟发现真相，制造永劫回归。
第五层：开拓者作为外来变量，和昔涟/德谬歌一起打破既定结局。
```

## 3. 黄金裔关系

```text
白厄：负世者，终局核心。
昔涟/德谬歌/迷迷：记忆、爱、第十三泰坦、闭环关键。
阿格莱雅：奥赫玛秩序与黄金裔组织者。
刻律德菈：律法火种，执棋的君主，城邦权力线。
海瑟音/海列屈拉：海洋火种，奏浪的剑骑，与刻律德菈形成“命令与归处”关系。
那刻夏：理性怀疑者，挑战神话叙事。
风堇：天空与医者线。
遐蝶：死亡/冥河线。
万敌：纷争与战斗线。
```

## 4. 刻律德菈与海瑟音补充

刻律德菈不能漏。她代表的是翁法罗斯中层政治：律法、君主、棋局、战争与统治。海瑟音也不能被当成背景角色，她是海洋火种的剑骑，与刻律德菈形成“君主给出方向，剑骑把命令变成归处”的关系。

如果只讲白厄和昔涟，翁法罗斯会变成纯终局轮回故事；补上刻律德菈和海瑟音后，才能看见城邦如何把英雄史诗变成制度和战争。

## 5. 主题

翁法罗斯真正的问题是：如果一个世界最初只是实验数据，但里面的人反复痛苦、反复选择、反复爱着彼此，他们算不算真正活过？答案是算。
