# 4.x 二相乐园：欢愉、幻造种、告死魔与姬子身份

## 1. 快速结论

二相乐园表面是欢愉与幻月游戏，底层是幻造种、愿力、创伤消费、丰饶污染、毁灭旧账和姬子身份问题。

## 2. 核心设定

```text
幻造种：由想象、绘画、愿力塑造出的生命。
幻月游戏：谒者通过愿力竞争，接近欢愉权能。
旧告死魔：十五年前血案源头。
满愿：幸存者变成强制幸福的加害者。
归寂：毁灭旧账的真正危险。
姬子：少女姬子与列车姬子的继承/身份问题。
```

## 3. 姬子线

二相乐园最狠的问题不是“列车姬子是不是假的”，而是“继承是否能成为真实”。如果列车姬子是少女姬子创造出的幻造种，她仍然用漫长旅途证明了自己是姬子。

## 4. 主题

这篇章把欢愉、创伤和身份混在一起：人们需要故事，媒体消费伤口，受害者被迫表演幸福，最后有人选择把全世界加工成幸福牢笼。
