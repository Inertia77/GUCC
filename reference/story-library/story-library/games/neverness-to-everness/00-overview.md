# 异环 / NTE 剧情总览

## 0. 快速结论

《异环 / Neverness to Everness》是超自然都市开放世界 RPG。它的核心不是传统末世神话，而是“现代都市日常与异常现象共存”。玩家作为无证上岗的异象猎人，从海特洛市开始，加入经营不善但接取民间异象委托的古董店「伊波恩」，和伙伴们处理城市中的怪奇事件。

## 1. 世界观关键词

```text
海特洛市：故事起点，异常成为新日常的都市。
异象：城市中出现的超自然异常事件。
异象猎人：处理、调查、摆平异常事件的人。
伊波恩古董店：主角所属据点，民间委托入口。
异能者/歧骸等超自然居民：让城市日常与怪奇混在一起。
```

## 2. 叙事基调

异环适合按“都市怪谈单元剧 + 主线谜团”理解：

- 单元剧负责展示城市生活和怪奇委托。
- 主线负责追问异象来源、城市管理结构、主角身份和伙伴过去。
- 日常搞笑与大型异常事件并存。

## 3. 与无限大的区别

异环是 NTE / Neverness to Everness，城市是海特洛市，核心是异象猎人和伊波恩古董店。  
无限大是 ANANTA，城市是 Nova City，核心是特殊小队/队长和更偏动作化、自由都市冒险的叙事。

这两个不要混。
