# 鸣潮 剧情总览

## 0. 快速结论

《鸣潮》的主线可以理解成“失忆神格/高维变量回到灾后世界，重新理解人与文明如何在鸣式灾害中延续”。表层是漂泊者醒来后游历各地，底层是关于残象、频率、共鸣、文明监测与世界级灾害的系统叙事。

主线阶段大致分为：

```text
今州篇：漂泊者苏醒，接触今州秩序、残象灾害、残星会。
乘霄山篇：今汐、岁主、时间/守护者与今州更古老的责任。
黑海岸篇：守岸人、数据海、世界监测组织与漂泊者过去。
黎那汐塔篇：新国度的艺术、信仰、贵族、面具与世界真相扩展。
后黎那汐塔篇：琥珀之城/废土晨曦，进入更广阔世界线。
```

## 1. 世界观关键词

- **悲鸣/鸣式灾害**：世界大灾变的核心机制，改变生态、文明与生命形态。
- **残象**：灾害后的怪物与频率异常体现。
- **共鸣者**：与特定频率/能力绑定的人。
- **漂泊者**：失忆的主角，和世界根源、文明监测、灾害处理有更深关系。
- **残星会**：早期主要敌对组织之一，围绕灾害、信仰、异化和世界目标展开行动。
- **黑海岸**：世界级监测与保护组织，守岸人线让漂泊者过去逐渐浮出。
- **黎那汐塔**：从东方灾后军事/守城叙事转向欧洲式艺术、宗教、贵族与仪式叙事。

## 2. 主题

《鸣潮》不是单纯“打怪救城”。它真正反复写的是：

```text
灾后文明如何维持秩序？
人类如何与不可理解的频率灾害共存？
一个被世界需要的存在，能不能拥有自己的选择？
守护世界是责任，还是被迫重复的命运？
```

漂泊者的故事核心不是“我是谁”这么简单，而是“我曾经为这个世界做过什么，现在还要不要继续承担”。
