# 1.0 今州开局：失忆漂泊者与残象灾害

## 快速结论

今州篇建立了《鸣潮》的基本语法：漂泊者失忆醒来，进入一个已经被鸣式灾害长期改变的世界。今州不是新手村，而是灾后前线城市。

## 剧情结构

漂泊者被秧秧、炽霞、白芷等人带入今州秩序，接触共鸣者、残象、前线防务和城市政治。残星会作为早期敌对势力登场，把灾害从“怪物袭击”提升为“有人主动利用灾害”。

## 人物关系

```text
漂泊者：失忆变量。
秧秧/炽霞/白芷：今州生活与行动入口。
忌炎：军队与前线压力。
今州高层：灾后秩序维护者。
残星会：灾害利用者与信仰式敌人。
```

## 主题

灾后文明不是靠热血活着，而是靠防线、信息、牺牲和日复一日的秩序维护活着。
