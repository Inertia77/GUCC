# 绝区零 剧情总览

## 0. 快速结论

《绝区零》的主线不是“都市打怪日常”，而是一个围绕新艾利都、空洞灾害、绳匠网络、Phaethon 身世和旧世界遗产展开的都市群像。它表面轻快，底层其实是灾后城市如何把异常变成产业、秩序和日常。

主线阶段：

```text
序章：Phaethon 兄妹、绳匠身份、空洞探索。
第一季：狡兔屋、白祇重工、维多利亚家政、卡吕冬之子、对空六课等阵营建立。
第二季：雅努斯区、新城区、新治安局/黑枝线，城市版图扩张。
第三季：罗斯凯利法、天之算枢、Phaethon 身世与 Helios 遗产继续推进。
```

## 1. 世界观关键词

- **空洞**：吞噬城市空间的灾害，也是资源、危险和产业来源。
- **以骸**：空洞中的异化威胁。
- **绳匠**：引导代理人进入空洞的特殊职业。
- **Phaethon**：主角兄妹过去的传奇绳匠身份。
- **新艾利都**：灾后仍能繁荣的都市，也是秩序、资本、治安和黑色委托交织之地。
- **邦布**：可爱外表下承担大量城市功能和空洞行动辅助的智能存在。

## 2. 叙事特点

绝区零不是大地图神话，而是“街区剧”。它最适合按阵营和案件理解：每个版本不是只推主线，而是通过一个组织、一个街区、一个委托，把新艾利都的城市结构再打开一层。

```text
狡兔屋：底层委托与生活感。
白祇重工：建设、事故、企业与工程伦理。
维多利亚家政：优雅外壳下的异常处理专业队。
卡吕冬之子：外环、车队、荒野秩序。
治安局：官方秩序与灰色手段。
对空六课：更高危的空洞/异常处理能力。
```

## 3. 主线核心问题

Phaethon 兄妹到底为什么重要？Helios 遗产是什么？空洞灾害背后是否存在旧文明/旧实验/旧组织？新艾利都表面把空洞管理成产业，但这套秩序是否建立在被掩盖的旧真相上？
